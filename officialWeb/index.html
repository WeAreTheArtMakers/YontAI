Powerful Features

YontAI is designed to take your local AI coding assistant to the next level.

⚡

MLX Apple Silicon Support

Up to 40% speed improvement on M1/M2/M3/M4 chips. Run 7B models on 16GB RAM with 4-bit quantization.

🔄

Multi-Model Orchestration

Smart routing: 1-3B fast model for FIM, 7-16B smart model for chat. Automatic switching, zero latency.

⌨️

FIM Code Completion

Fill-in-the-Middle with DeepSeek-Coder format. Analyzes prefix/suffix context. <150ms response time.

🔍

RAG Context Engine

AST parsing in 17 languages with tree-sitter. ChromaDB vector search. Smart context with dependency graph.

🌐

Web Knowledge Ingestion

GitHub API, npm, PyPI integration. One-click code fetching, cleaning, embedding, and RAG indexing.

🧪

AI Lab Model Training

LoRA/QLoRA fine-tuning with MLX. Dataset builder for instruction, FIM, and chat format training data.
